About This Project

StudyTracker is a simple web project developed for students who want to organize their tasks and manage their study time more effectively.

The project was built using HTML, CSS, and JavaScript. HTML is used to create the page structure, CSS is used for the visual design, and JavaScript is used to add dynamic functionality such as task creation, deletion, and localStorage support.

The main aim of this website is to provide a user-friendly and clean interface for daily task tracking. It also demonstrates basic front-end web development skills in a practical way.
